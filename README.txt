Project: Expense Tracker Mobile Application

Overview: 
This dataset includes user-generated data collected through an Expense Tracker mobile application. It contains information about users, budgets, transactions, categories, and alerts. 
The dataset is intended for financial management purposes, enabling users to track spending, set budgets, and receive alerts for overspending.


Backend Files: app.py, myenv(folder), requirements.txt, MySQL(Folder)

FrontEnd Files: Alert.swift, Budget.swift, Category.swift, Transaction.swift, User.swift, AlertService.swift, BudgetService.swift, CategoryService.swift, Constants.swift, NetworkManager.swift, TransactionService.swift, UserService.swift, AlertView.swift, BudgetsView.swift, BudgetCardView.swift, CategoryView.swift, TransactionFormView.swift, UserDetailsTabView.swift, ContentView.swift

How to execute: 
Start your backend - cd Backend
1. source myenv/bin/activate 
2. Install dependecies - pip install -r requirements.txt
3. python app.py
4. Go to xcode - click run button or command + r if on Mac
